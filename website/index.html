How it works

From setup to first review in four steps

Connect your repository once. After that, every pull request is reviewed automatically.

Step 1

Set up your AI

Add your preferred language model in Settings — one profile is enough to get going.

Step 2

Connect a repository

Link your Git provider and credentials so Nexo Co-Review can receive events and post feedback.

Step 3

Enable the webhook

Point pull request events from your Git provider to your instance — one setup, then reviews run on their own.

Step 4

Open a pull request

Watch the review run in the dashboard. Findings appear as soon as the review completes.
